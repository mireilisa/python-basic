# Monthly Budget Calculator

A command-line Python application for tracking and analyzing a monthly personal budget.

The program collects income and expenses by category, calculates totals and balance, and provides visual warnings based on budget status.

---

## Key Features

- Monthly income input
- Expense tracking by category
- Automatic calculation of:
  - Total expenses
  - Remaining balance
- Budget status alerts:
  - ❌ Over budget
  - ⚠️ Budget is tight
  - ✅ Budget under control
- Colored terminal output for better readability

---

## Tech Stack

- Python 3
- CLI (Command Line Interface)
- Functions and modular structure
- Lists and loops
- Conditional logic
- ANSI escape codes for colored output
